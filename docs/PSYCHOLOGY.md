# 心理学分析功能指南

## 功能概述

基于群聊消息的发言频率、互动模式、情绪表达等数据，提供心理学层面的分析和洞察。

---

## 分析维度

### 权重说明

| 维度 | 权重 | 说明 |
|------|------|------|
| **用词分析** | 40% | 核心分析，反映思维模式和情绪状态 |
| **互动模式** | 30% | 核心分析，反映人际关系 |
| **发言频率** | 15% | 辅助参考，不过度解读 |
| **响应速度** | 15% | 辅助参考，不过度解读 |

> **重要**: 发言频率和响应速度仅作为辅助参考，避免简单贴标签。重点分析用词和互动模式。

---

### 1. 用词分析 (核心，40%)

#### 词库分类

| 类别 | 示例词 | 心理学含义 |
|------|--------|------------|
| **情绪词** | 开心/累/烦/爽/郁闷/焦虑 | 情绪表达强度和频率 |
| **肯定词** | 好的/没问题/可以/行/OK | 配合度、积极性 |
| **否定词** | 不行/不要/算了/随便/懒得 | 消极态度、抵触情绪 |
| **疑问词** | 为什么/怎么/什么/如何 | 好奇心、探索欲 |
| **绝对词** | 一定/必须/绝对/永远/所有 | 完美主义、固执倾向 |
| **模糊词** | 可能/也许/大概/随便/看着办 | 犹豫不决、回避态度 |
| **第一人称** | 我/我的/我自己 | 自我关注度 |
| **我们称谓** | 我们/咱们/大家一起/团队 | 团队意识、归属感 |

#### 用词指标

| 指标 | 计算方法 | 含义解读 |
|------|----------|----------|
| **肯定词占比** | 肯定词数/总词数 | 高 (>25%)：积极配合；低 (<15%)：消极抵触 |
| **否定词占比** | 否定词数/总词数 | 高 (>15%)：消极/抵触；正常 (5-10%) |
| **情绪词占比** | 情绪词数/总词数 | 高 (>20%)：情绪化；正常 (5-15%)；低 (<5%)：理性克制 |
| **"我"vs"我们"** | 我次数/我们次数 | 高 (>2:1)：自我中心；低 (<1:2)：团队导向 |
| **绝对词频率** | 绝对词数/天 | 高 (>10 次)：完美主义/固执；正常 (<5 次) |
| **疑问词频率** | 疑问词数/天 | 高 (>20 次)：好奇心强；低 (<5 次)：被动接受 |

### 典型语录

> "大家一起努力" (团队导向)

### 2. 互动对象分析 (核心，30%)

| 指标 | 计算方法 | 心理学含义 |
|------|----------|------------|
| **单向高频** | 回复某人>70% | 关系密切，或可能存在依赖/讨好倾向 |
| **多向平衡** | 回复分散 (<30% 每人) | 社交平衡、健康关系 |
| **无互动** | 只发言不回复 | 自我中心、缺乏社交意识 |

---

### 3. 发言频率 (辅助，15%)

| 指标 | 计算方法 | 含义 (仅供参考) |
|------|----------|----------------|
| **高频** | >50 条/天 | 可能表现欲强，或工作性质需要 |
| **中频** | 10-50 条/天 | 正常社交参与 |
| **低频** | <10 条/天 | 可能内向，或工作忙碌 |

> **注意**: 发言频率受工作性质、性格、时区等多因素影响，不可单独作为判断依据。

---

### 4. 响应速度 (辅助，15%)

| 指标 | 计算方法 | 含义 (仅供参考) |
|------|----------|----------------|
| **快速** | <5 分钟 | 可能重视对话，或恰好在线 |
| **正常** | 5-30 分钟 | 健康响应模式 |
| **慢速** | >30 分钟 | 可能忙碌，或不重视 |

> **注意**: 响应速度受工作环境、会议、时差等影响，不可过度解读。

## 配置说明

### 启用心理分析

```json
{
  "psychologicalAnalysis": {
    "enabled": true,
    "reportFrequency": "weekly",
    "includeInProfile": true,
    "alertThresholds": {
      "highFrequency": 50,
      "lowFrequency": 2,
      "unbalancedInteraction": 0.7
    }
  }
}
```

### 配置项说明

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `enabled` | 是否启用心理分析 | `true` |
| `reportFrequency` | 报告频率 | `"weekly"` |
| `includeInProfile` | 是否包含在人物档案中 | `true` |
| `alertThresholds.highFrequency` | 高频发言阈值 (条/天) | `50` |
| `alertThresholds.lowFrequency` | 低频发言阈值 (条/天) | `2` |
| `alertThresholds.unbalancedInteraction` | 不平衡互动阈值 | `0.7` |

## 使用场景

### 工作群

**适用**: 团队管理、员工关怀、冲突预警

**示例**:
```
【周度心理报告】

🔴 关注预警:
- 王五：本周发言仅 3 条 (上周 25 条)，可能情绪低落
- 张三：回复李四占比 85%，可能存在小团体现象

💡 管理建议:
- 主动关心王五，了解是否有工作压力
- 组织团建活动，促进团队融合
```

### 朋友群

**适用**: 关系洞察、聚会策划

**示例**:
```
【朋友群关系分析】

👥 核心人物：张三 (发言最多，组织力最强)
🤝 亲密关系：张三 ↔ 李四 (互动占比 60%)
🔍 边缘人物：王五 (发言少，互动少)

💡 聚会建议:
- 让张三组织，参与度高
- 主动邀请王五，避免边缘化
```

### 家庭群

**适用**: 家庭关系、老人关怀

**示例**:
```
【家庭群关怀提示】

👴 爷爷：本周发言 0 条 (平时每天 5-10 条)
   → 建议电话关心身体状况

👦 小明：发言频率下降 50%
   → 可能学习压力大，建议沟通
```

## 隐私与伦理

### 使用原则

1. **知情同意**: 告知群成员有心理分析功能
2. **数据最小**: 只分析必要数据，不存储原始消息
3. **用途限制**: 仅用于关怀和改善，不用于操控
4. **保密原则**: 心理报告不公开传播

### 禁用场景

❌ 用于操控他人行为
❌ 用于职场歧视/偏见
❌ 用于恶意揣测/传播
❌ 用于商业目的未经同意

## 常见问题

### Q: 心理分析准确吗？

A: 基于统计学和行为心理学研究，但仅供参考，不能替代专业心理评估。

### Q: 可以关闭吗？

A: 可以，配置中设置 `"enabled": false` 即可。

### Q: 数据会保存多久？

A: 默认 180 天，可在配置中修改 `retentionDays`。

### Q: 可以只分析部分人吗？

A: 可以，在 `watchMembers` 中指定成员列表。
